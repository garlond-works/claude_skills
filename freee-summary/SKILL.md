# freee 月次サマリー作成

## 必要な情報を確認

以下を確認する（提供済みならスキップ）：
1. 対象月（例：2026年3月）
2. データ取得方法：freee MCP で直接取得 or CSVファイルのパスを指定
3. 報告先（KT税理士法人 / 自己管理 / 確定申告用）

## freee MCP でデータ取得する場合

freee MCP が接続済みであれば直接取得できる：
- 売上：`freee_api_get` で請求書・売上データを取得
- 経費：`freee_api_get` で経費・支払データを取得
- 事業所ID：10863824（ガーロンドワークス）

## ⚠️ Gotchas（やりがちな失敗）

**消費税込みで集計する**
集計は必ず税抜きで行う。freeeのデータは税込み表示になっている場合があるため注意。

**クライアント名をフルネームで書く**
KT税理士法人・H社・SDTはイニシャル表記で統一。対外資料には使用しない（内部管理用）。

**未収・未払いを含めて集計する**
請求済みでも未入金のものは売上に含めない（入金ベースで集計）。freee上のステータスを確認する。

**freee MCPのトークン期限切れ**
アクセストークンは6時間で失効する。エラーが出たら `freee_authenticate` で再認証する。
